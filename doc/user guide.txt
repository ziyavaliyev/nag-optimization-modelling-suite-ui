1. download correct version dcl6i37ngl_v370 https://www.nag.com/content/downloads-dco-c-dcl6i37ngl
		            nll6i285bl      https://www.nag.com/content/downloads-nag-library-nll6i285bl		            
they are in tar archive. extract then run 
./install.sh
(yes, accept, return (for default install dir))

2. acquire git repo
   git clone https://git.rwth-aachen.de/tranmankhang1705/nag-optimization-modelling-suite-ui.git


3. place nag.key from "license" in
      $HOME/nag.key
   or
      /opt/NAG/nag.key
   or
      /usr/local/NAG/nag.key

4. From anywhere, run commands:

      mkdir build && cd build

      cmake ${REPO_DIR} -DNAG_dco_cpp_DIR=${HOME}/NAG/dcl6i37ngl_v370/ -DNAG_Library_DIR=${HOME}/NAG/nll6i285bl/
   (replace ${REPO_DIR} with git repo directory, ${HOME} with /home/your_username/) 
   
      make 
   (this compiles all)

      ./examples/examples.xxx

   To compile specific examples:

      make examples.xxx

      ./examples/examples.xxx

5. To add your own program:
      Write your .cpp file
      Make a subfolder within ${REPO_DIR}/examples/
      Place .cpp program within this subfolder.
      Repeat step 4, but now do make examples.name_of_your_subfolder
      ./examples/examples.name_of_your_subfolder
